# Contributing

Thank you for contributing to this project.

## AI-Assisted Contributions

AI tools and agents may be used during development. However, the following rules apply:

- Pull requests must be created and submitted by a human. Automatically created pull requests from AI agents are not allowed.
- All AI-generated or AI-modified changes must be carefully reviewed and thoroughly tested by a human.
- AI agents must not post comments, replies, or other messages on pull requests. All communication must be written and submitted by a human.
- If an AI tool or agent was used, its name must be disclosed in the pull request description.
